Esta clase se encarga de obtener los nombres de los "Scenarios" de prueba
encontrados en las features. Con esto podemos sacar la cantidad exacta de test por feature.
Es solo para obtener un numero de casos de pruebas automatizados dentro de este proyecto.
Debes ingresar el "path" de la feature que deseas obtener los datos y esta los obtiene.

Posee su propio ejecutor para independizar esta actividad de las pruebas.
